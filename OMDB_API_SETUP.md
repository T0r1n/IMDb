# Настройка OMDb API для отображения постеров фильмов

## Проблема
По умолчанию постеры фильмов не отображаются, показывается только текст с градиентным фоном.

## Решение
Используйте бесплатный OMDb API для получения реальных постеров фильмов.

## Инструкция по настройке:

### 1. Получите бесплатный API ключ:
1. Перейдите на сайт: http://www.omdbapi.com/apikey.aspx
2. Выберите "FREE" (1000 запросов в день)
3. Заполните форму регистрации
4. Проверьте email и подтвердите регистрацию
5. Скопируйте полученный API ключ

### 2. Настройте ключ в системе:

#### Вариант A: Через переменную окружения (рекомендуется)
```bash
# Windows PowerShell
$env:OMDB_API_KEY="ваш_ключ_здесь"

# Windows CMD
set OMDB_API_KEY=ваш_ключ_здесь

# Linux/Mac
export OMDB_API_KEY="ваш_ключ_здесь"
```

#### Вариант B: Прямо в коде (менее безопасно)
Откройте файл `server/main.py` и найдите строку:
```python
omdb_api_key = os.environ.get('OMDB_API_KEY', '')
```

Замените на:
```python
omdb_api_key = os.environ.get('OMDB_API_KEY', 'ваш_ключ_здесь')
```

### 3. Перезапустите Flask сервер:
```bash
python server/main.py
```

### 4. Обновите страницу в браузере:
Нажмите Ctrl+F5 для полной перезагрузки.

## Альтернативные варианты:

### TMDB API (тоже бесплатный)
1. Зарегистрируйтесь на https://www.themoviedb.org/
2. Получите API ключ в настройках аккаунта
3. Установите переменную окружения `TMDB_API_KEY`

## Проверка работы:
После настройки API ключа, откройте консоль браузера (F12) и проверьте логи.
Вы должны увидеть успешные запросы к API и реальные URL постеров.

## Ограничения:
- **OMDb FREE**: 1000 запросов в день
- **TMDB**: Без ограничений для личного использования
